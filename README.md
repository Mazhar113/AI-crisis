# Crisis Response AI Map - Ultimate Live Version

## Features
- Live data feeds: NDMA, USGS, OpenWeather, Volunteer submissions
- AI-powered classification (10 categories)
- Reliability scoring with explanations
- Duplicate detection & merged counts
- Admin dashboard with JWT authentication
- Real-time updates (WebSockets)
- OpenStreetMap frontend
- Dockerized deployment
- Ready for Render (backend) & Vercel (frontend)

## Run Locally
Make sure Docker is installed.

```
docker-compose up --build
```

Frontend: http://localhost:3000  
Backend API: http://localhost:8000
